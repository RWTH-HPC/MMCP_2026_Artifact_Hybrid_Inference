This directory should contain the following files:
- `grid_les_medium.hdf5`: computational grid comprising 732x131x250 cells
- `properties_run_les_ref_medium.toml`: solver input configuration file
- `restart_les_init_medium.hdf5`: restart file used to initialize the TBL simulation
- `transformer_inference_scripted_fw2.pt`: Transformer model

These files take up to 1GB of storage, so we made them available via [Sciebo](https://rwth-aachen.sciebo.de/s/yxSDGncnbjcA4Tk), please download!
